# Portfolio Website - Rafif Firdausy Ahla

### Short Description
A personal portfolio website showcasing my journey as a Computer Science student at BINUS University, my technical projects (Arduino & Laravel), and my competitive achievements in Informatics and Mathematics.

### Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Deployment:** Vercel / Netlify

### Features
* **Responsive Design:** Fully accessible on mobile, tablet, and desktop.
* **Dark/Light Mode:** Saves user preference using LocalStorage.
* **Interactive UI:** Smooth scrolling and animated hamburger menu.
* **Project Highlights:** Details on "Aespark" (Laravel) and "Smart City RFID" (Arduino).
